## Problem

As specialized AI agents proliferate — managing calendars, researching suppliers, executing purchases — they need to hire, pay, and contract with each other autonomously. AI shopping agents also have no standardized way to hold funds or build verifiable transaction histories across merchants. There is no open protocol for agents to discover services, agree terms, and settle payments without routing every transaction through a human approval loop.

## Solution

Open protocols for AI agents to publish capabilities, negotiate service terms, execute purchases, and settle micropayments autonomously — creating a machine-to-machine service economy where agents from any organization can transact as fluidly as APIs call each other today.

## Why Ethereum

Composability: a shared settlement layer with open standards lets agents from any organization transact without pre-negotiated bilateral integrations — the agent economy becomes interoperable by default rather than requiring every agent developer to build custom payment bridges.
